# client-hub

Aplicação onde é possível se cadastrar, fazer lolgin e registrar clientes.

## Instalação backend:

```bash
$ npm install
```

## Rodando a aplicação:

```bash
# development
$ npm run start:dev
```

## Instalação frontend: 

```bash
$ npm install
```

## Rodando a aplicação:

```bash
# development
$ npm run start
```
